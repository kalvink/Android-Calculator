# Android-Calculator
Basic Calculator coded in Java converted into an Android app by Kalvin Kao
https://play.google.com/store/apps/details?id=com.kalvinkao.calculator

Functionality
- Addition
- Subtraction
- Multiplication
- Division
- Equate
- Decimals
- Clear all
- Back space

Tools used
- Android Studio
